Разом із файлами програми надано список таких файлів:

biathlon input.txt - тестовий файл з коректною інформацією

biathlon output.txt - файл з результуючою інформацією коректного файлу

empty.txt - тестовий файл без записів, лише з ключовими словами

error footer 1.txt - тестовий файл з помилковою інформацією в footer'і (кількість записів)

error footer 2.txt - тестовий файл з помилковою інформацією в footer'і (кількість промахів)

error header.txt - тестовий файл з помилковою інформацією в header'і (кількість етапів)

time fail.txt - тестовий файл зі штрафним часом, більшим за сумарний (рядок №5)

two same numbers.txt - тестовий файл з різними учасниками, що мають однаковий стартовий номер (рядок №11)

two same entries.txt - тестовий файл з двома однаковими записами (рядок №9)

syntax footer.txt - тестовий файл з помилкою в синтаксисі footer'а

syntax header.txt - тестовий файл з помилкою в синтаксисі header'а

syntax error entry.txt - тестовий файл з помилкою в синтаксисі (рядок №3)

wrong entry arguments quantity.txt -  - тестовий файл з помилкою в кількості переданих аргументів (рядок №3)
